函数。

结束。

你知道：

```

struct
```

因为：

# 第四节：为什么课程不从变量开始？

有人可能疑惑：

为什么第一课讲：

历史。

哲学。

工程。

而不是：

变量。

------

原因：

因为：

如果不知道：

为什么。

学习：

会变成：

记忆。

------

例如：

今天：

学：

interface。

如果不知道背景：

你会觉得：

```
type Reader interface {
    Read()
}
```

奇怪。

------

但是知道：

Go：

追求：

松耦合。

组合。

维护。

你会理解：

interface 为什么这样设计。

------

知识顺序：

应该是：

```
为什么

↓

设计思想

↓

语法

↓

使用

↓

工程实践
```

------

不是：

```
语法

↓

背下来

↓

忘记
```
